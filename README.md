# ssl-odds-pages

本仓库用于归档和公开发布多个独立的 HTML 页面。这些页面不针对任何特定域名、网站或服务，仅作为通用页面资源的存储与分发用途。

## 项目简介

ss-odds-pages 是一个集中存放静态 HTML 页面的仓库。所有页面均为独立、自包含的 HTML 文件，不依赖外部动态服务或数据库。适合用于存档、展示、测试或临时页面引用等场景。

## 目录说明

```
/
├── README.md          # 本文件
├── pages/             # 页面归档目录
│   ├── example1.html  # 示例页面
│   └── example2.html
└── assets/            # 公共资源（样式、脚本、图片等）
    ├── style.css
    └── script.js
```

- `pages/`：存放所有独立 HTML 页面文件，每个文件可直接通过浏览器打开。
- `assets/`：存放页面可能共用的静态资源文件。

## 页面归档说明

- 所有页面均为静态 HTML，不含服务器端脚本。
- 页面之间相互独立，不共享状态或数据。
- 归档内容可能随时间更新或移除，不保证长期完整可用。
- 页面内容仅用于归档展示，不代表任何立场或推荐。
